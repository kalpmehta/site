Magento get State Code, ID, Name from Region ID

2014-06-25 · kalpesh

Magento get state name, ID, code from region ID or region code.

Get state code from state id

{code type=php}$region = Mage::getModel(‘directory/region’)->load(12);
$state_code = $region->getCode(); //CA{/code}

Get state name from state id

{code type=php}$region = Mage::getModel(‘directory/region’)->load(12);
$state_name = $region->getName(); //California{/code}

Get state id from state code

{code type=php}$region = Mage::getModel(‘directory/region’)->loadByCode(‘CA’, ‘US’);
$state_id = $region->getId(); //12{/code}
