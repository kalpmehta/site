Magento get store information

2013-06-29 · kalpesh

Magento get store information from any page.

{code type=php}$store = Mage::app()->getStore(); //Mage_Core_Model_Store object{/code}

Now to get store ID,
{code type=php}$storeID = $store->getStoreId();{/code}

To get store code,
{code type=php}$storeCode = $store->getCode();{/code}

To get store Website ID,
{code type=php}$websiteID = $store->getWebsiteId();{/code}

To get store name,
{code type=php}$storeName = $store->getName();{/code}

To get Default View store name,
{code type=php}$storeName = $store->getFrontendName();{/code}

To check if store is active or not,
{code type=php}$active = $store->getIsActive();{/code}

To get store’s Home URL,
{code type=php}$storeHomeURL = $store->getHomeUrl();{/code}

If you want to find store’s group name,
{code type=php}$storeGroupName = $store->getGroup()->getName();{/code}

If you want ADMIN Store information, admin store has ID 0. So, you can get the details of Admin Store like this:
{code type=php}$store = Mage::getModel(‘core/store’)->load(0);{/code}

and then getting all the information as shown above.
